# v1.0.0 — Lorsque le rideau se soulève

Pendant longtemps, le projet n'a été qu'une succession de plans, de schémas et d'intentions.

Une palette de couleurs sans écran.
Des pages sans visiteurs.
Des idées sans visage.

Aujourd'hui, quelque chose apparaît.

Les fondations sont toujours là, invisibles sous la scène.
Mais au-dessus, les premiers éléments prennent forme : une voix, une direction, une présence.

Le chantier continue.

Simplement, le rideau n'est plus complètement fermé.
