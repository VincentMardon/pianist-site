# Pianist Site

Site vitrine personnel de Vincent Mardon, pianiste accompagnateur indépendant.

## Objectifs

Ce projet poursuit plusieurs objectifs :

* Disposer d'une présence en ligne simple et professionnelle.
* Présenter mes prestations de pianiste accompagnateur.
* Permettre une prise de contact rapide.
* Expérimenter la maintenance d'un site réel sur le long terme.
* Servir de terrain d'expérimentation pour de futurs composants réutilisables.

## Contenu

Le contenu principal est rédigé en Markdown afin de privilégier la simplicité de mise à jour.

Pages prévues :

* Accueil
* Présentation
* Tarifs
* Contact

## État du projet

Projet en cours de construction.

Version actuelle : prototype minimal.

## Technologies envisagées

* Markdown
* React / Next.js
* TypeScript
* Vanilla Extract

## Notes

Ce dépôt privilégie la clarté du contenu avant la sophistication technique.

Les composants réellement réutilisables pourront être extraits ultérieurement vers un dépôt dédié de type "design system".
