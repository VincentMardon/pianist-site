# Tarifs

Les tarifs ci-dessous donnent un cadre général pour les prestations les plus courantes.

Chaque demande peut cependant varier selon le niveau de préparation nécessaire, la durée de l’intervention, le lieu, le type de structure et les objectifs du projet. Pour les accompagnements réguliers, les projets associatifs, pédagogiques ou les interventions longues, un devis personnalisé peut être proposé.

## Accompagnement

| Prestation              |     Tarif |
| ----------------------- | --------: |
| Accompagnement ponctuel |    60 €/h |
| Demi-journée            | Sur devis |
| Journée                 | Sur devis |

Le tarif horaire d’accompagnement peut être adapté selon le type de structure, la nature du projet, la durée de l’intervention et la régularité de la collaboration.

Des forfaits à la demi-journée ou à la journée peuvent être proposés pour les répétitions longues, stages, examens, auditions, concerts, projets pédagogiques ou événements spécifiques.

## Cours et accompagnement pédagogique

| Prestation                  |       Tarif |
| --------------------------- | ----------: |
| Cours débutant — 30 minutes |        20 € |
| Cours particulier — 1 heure | 40 à 60 €/h |

Les cours sont construits selon le niveau, les objectifs et le degré d’autonomie de l’élève.

Pour les débutants, un format de 30 minutes peut être plus adapté, notamment au départ, afin de garder une séance claire, concentrée et efficace. Pour les élèves plus avancés, les reprises sérieuses, le travail d’interprétation ou la préparation d’une œuvre, le tarif est ajusté selon le niveau d’exigence et le travail demandé.

## Concerts et projets spécifiques

| Prestation                                               |     Tarif |
| -------------------------------------------------------- | --------: |
| Concerts, projets artistiques, interventions spécifiques | Sur devis |

Les concerts, créations, projets collectifs, interventions exceptionnelles ou demandes nécessitant une préparation particulière font l’objet d’un échange préalable et d’un devis adapté.

## Déplacements

| Prestation        | Tarif |
| ----------------- | ----: |
| Déplacement local | + 3 € |

Les frais de déplacement peuvent être ajustés selon la distance, les horaires et les conditions d’intervention.
